# issue INSERTモード中のfetchを廃止するか検討する #17
[issues #17](https://github.com/cat2151/voicevox-playground-tui/issues/17)

# 廃止する場合
- ENTERまたはESCで行を確定したときにその行をfetch、となる
- メリット、fetchの仕様がシンプルになる

# 廃止しない場合
- ENTERまたはESCで行を確定する前から、fetchが行われる
- メリット、特に短い文章で、ENTERした直後から演奏が行われて、体験がよい
    - 実績、実際、体験がよいのを検証済み

# 結論
- 廃止しない。引き続きfetchをする
