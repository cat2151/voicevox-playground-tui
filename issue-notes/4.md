# issue TUIでマウスを使ったイントネーション編集 #4
[issues #4](https://github.com/cat2151/voicevox-playground-tui/issues/4)

# メリット
- イントネーション編集はマウスとの相性がよい、特に「このモーラのピッチは、こことここの中間くらいがいい」、は、マウス（より適切にはタッチペン等）が適している、と判断する

# 擬似折れ線グラフの表示
- 文字で表現できる範囲での表示とする
- 0.1を1行、として表示する
- つまり上下幅が1.0あれば、10行使って表示する
- 画面の行数に対して、上下幅が不足している行があれば、その行は「グレーアウト表示し、編集不可」とする

# マウスdown時の挙動
- その近傍が、モーラのどの列に該当するか？を判定する
- その列について、pitchを、「マウスdownした場所そのもの」として決定する
   - 例えば画面上で、pitch5.5を示す場所、にマウスがあれば、pitchが5でも6でも、5.5に瞬間的に移動とする
