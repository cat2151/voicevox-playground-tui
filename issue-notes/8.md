# issue キーボードによる簡易イントネーションエディタモードをつける #8
[issues #8](https://github.com/cat2151/voicevox-playground-tui/issues/8)


- normal mode v を仮で、イントネーション編集モードにする
    - ※このモードは別画面レイアウトである
    - レイアウト：
        - 1行目 : modeを表示する（イントネーション編集モード）
        - 2行目 : 本文
        - 3行目 : APIで取得されたモーラをspace区切りで列挙する（仮仕様）
        - 4行目 : モーラごとのpitchを数値表示する、整数部と小数部は1桁ずつ（仮仕様）
    - keybinds：
        - a～z : aがモーラ先頭のpitchを0.1 upする（仮仕様）
        - A～Z : Aがモーラ先頭のpitchを0.1 downする（仮仕様）
        - 1-9,0 : 数値直接入力モード。emacsライクな行入力で、ENTERで確定してplay（仮仕様）
        - ESC or ENTER : イントネーションを確定し、normal mode画面に戻る
        - ※a-zA-Zは、1secのデバウンスで、wavを生成してplayする
    - ※1行の途中でspeakerとstyle変更が発生して、fetch分割が行われた場合は、イントネーション編集不可であることを表示する（仮仕様。対応すると仕様が複雑なので）
- イントネーション編集有無、イントネーション内容、を行ごとに保持する
    - イントネーション編集有無は、セリフ変更でfalse。イントネーション編集してイントネーション確定でtrue。
- cache仕様を変更し、「行textと、イントネーション編集有無と、イントネーション内容」をkeyとし、wavデータをvalueとする。
- normal modeの左のレイアウトを変更し、「イントネーション編集有無フラグ」も「cache有無表示」の近くに表示する。
