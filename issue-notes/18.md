# issue NORMALモードでzmは折りたたみ、zrは折りたたみを開く、とする #18
[issues #18](https://github.com/cat2151/voicevox-playground-tui/issues/18)


- vimに似せたものである。vimはもっと複雑だが、割り切ってここではシンプルにする
- 折りたたみの基準は、行の先頭の半角spaceの有無とする。割り切ってvimよりシンプルにする
    - 折りたたみすると、行頭に半角spaceのある行が折りたたまれて見えなくなり、
        - そのぶん「行頭に半角spaceのない行」が上にズレて表示される
        - 折りたたみされているかは、「折りたたみされた行グループの1つ上の、非折りたたみ行」が、
            - 行番号の左に「+」が表示されていることでわかる
                - つまり折りたたみした瞬間、表示列が1列増える
- メリット、行数が多いセリフデータを読みやすくできる。セリフをグルーピングして読みやすくできる
