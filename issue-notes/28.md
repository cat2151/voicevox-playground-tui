# issue VOICEVOXエンジンが起動していないときは自動起動させる #28
[issues #28](https://github.com/cat2151/voicevox-playground-tui/issues/28)

# 何が困るの？
- vptクライアントをさっと実行したときにサーバー起動していなくて、vptクライアントが使えず、userが混乱、UX低下

# userがほしいものは？
- vptクライアントをさっと実行したときにサーバー起動していなければサーバーが自動起動し、そのままvptクライアント（voicevox-playground-tuiアプリのこと）が使える

# 注意点
- voicevox-playground-tuiが終了しても、起動したVOICEVOXエンジンのほうは継続してサーバーが生きているようにすること
