# issue 1画面に収まらない行数のとき、NORMALモードのoとOが表示崩れした #24
[issues #24](https://github.com/cat2151/voicevox-playground-tui/issues/24)

# 事象
- oとOを押したときの現在行まで水色になってしまった

# 分析
- おそらく1画面に収まらない行数のときの考慮漏れがある
  - oとOを押したときの行挿入が1画面に収まらないときに発生するscrollの考慮漏れ？

# userがほしいもの
- 1画面に収まらない行数のとき、oとOを押しても、表示崩れしない
  - oとOを押した直前の行は、水色にならず、通常色のまま
  - 「oとOを押した直後にinsertされ現在編集中の行」だけ水色になる
